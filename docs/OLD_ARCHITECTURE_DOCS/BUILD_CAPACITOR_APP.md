## 📋 Prerequisites

### Required: Java 21

The Capacitor Android build requires **Java 21**. You currently have Java 17 installed.

**Option 1: Install Java 21 (Recommended)**

1. Download Java 21 from:
   - Microsoft Build of OpenJDK: https://learn.microsoft.com/en-us/java/openjdk/download
   - Or Oracle JDK 21: https://www.oracle.com/java/technologies/downloads/#java21

2. Install Java 21

3. Update `android/gradle.properties`:
   ```properties
   org.gradle.java.home=C:\\Program Files\\Microsoft\\jdk-21.x.x-hotspot
   ```
   (Replace with your actual Java 21 installation path)

**Option 2: Temporarily Use Java 17 (Quick Fix)**

Edit `android/app/capacitor.build.gradle` and change:
```gradle
sourceCompatibility JavaVersion.VERSION_21
targetCompatibility JavaVersion.VERSION_21
```
to:
```gradle
sourceCompatibility JavaVersion.VERSION_17
targetCompatibility JavaVersion.VERSION_17
```

⚠️ **Note**: This file is auto-generated by `npx cap sync`, so you'll need to reapply this change after each sync.

## 🏗️ Building the APK

### Step 1: Build Next.js (Already Done ✅)

The build script (`scripts/build-capacitor.ps1`) has already:
- ✅ Built Next.js static export
- ✅ Synced to Capacitor
- ✅ Updated Android plugins

### Step 2: Build Android APK

Once Java 21 is installed, run:

```powershell
cd android
.\gradlew assembleDebug
```

This creates a debug APK at:
```
android/app/build/outputs/apk/debug/app-debug.apk
```

### Step 3: Build Release APK (Optional)

For a release build (requires signing configuration):

```powershell
cd android
.\gradlew assembleRelease
```

## 📱 Installing the APK

### Via ADB (Android Debug Bridge)

```powershell
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

### Manual Installation

1. Transfer `app-debug.apk` to your Android device
2. Enable "Install from Unknown Sources" in device settings
3. Open the APK file and install

## 🔄 Rebuilding After Changes

To rebuild after making code changes:

```powershell
powershell -ExecutionPolicy Bypass -File "scripts\build-capacitor.ps1"
cd android
.\gradlew assembleDebug
```

## 📝 Build Script Details

The `scripts/build-capacitor.ps1` script:
1. Temporarily moves API routes (not needed for static export)
2. Cleans `.next` build cache
3. Builds Next.js with `BUILD_MODE=capacitor` (static export)
4. Syncs web assets to Capacitor Android project
5. Restores API routes

## 🐛 Troubleshooting

### "Cannot find Java 21"
- Install Java 21 and update `gradle.properties` with the correct path

### "Build failed with API route errors"
- The build script should handle this automatically
- If issues persist, manually delete `app/api` before building

### "Gradle daemon issues"
- Stop Gradle daemon: `.\gradlew --stop`
- Clear Gradle cache: Remove `android/.gradle` directory
